# DecisionDeck

DecisionDeck is a smart decision-making web application that helps users evaluate choices using weighted scoring, confidence analysis, and intelligent insights.

## 🚀 Features
- Compare two options
- Add custom decision factors
- Weighted scoring system
- Confidence percentage
- Smart explanation of results

## 🧠 How It Works
Users input two choices and define factors like salary, growth, and stability.  
Each factor is assigned a weight and scores, and the app calculates the best option using a structured approach.

## 🛠 Tech Stack
- React.js
- JavaScript
- CSS

## 🎯 Use Case
Helps students and professionals make structured decisions like:
- Internship vs Placement
- Startup vs Corporate job
- Study vs Side hustle
